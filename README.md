# ds-helper

帮助你生成ds的模板代码，提高开发效率

# 安装

- https://marketplace.visualstudio.com/items?itemName=mathon.code-novel

# 使用

-  选中你需要生成模板的ds代码
- `alt + d`： 打开小说。(Mac：`cmd + alt + ,`)
- `vscode` 底部状态栏有跳页按钮和切换书籍及显示阅读进度的按钮
